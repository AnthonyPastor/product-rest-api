# Product Management REST API

API link: https://ap-product-nest-api.onrender.com/api

## Technologies
- NestJs
- Mongo
- Cloudinary
- JWT

## This API manage
-   Users
    -   Register
    -   Login
    -   Roles
 
-   Products
    -   Categories
    -   Prices
    -   Stock
    -   Searches
    
-   Payments (paypal, on test version it's a **FAKE** payment)
-   Admin actions
    -   Users management
    -   Product management
    -   Dashboard data
